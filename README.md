# Scientific Calculator in C++

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Scientific functions (square root,factorial,trigonometric function(sin , cosine , tangent)
- Support for parentheses to enforce order of operations
- User-friendly command-line interface

## How to Compile
To compile this project, you'll need a C++ compiler like g++. Follow these steps:
1. Open your command-line interface.
2. Navigate to the project directory.
3. Run the following command:
   ```bash
   g++ -o calculator main.cpp
   ```

## How to Use
After compiling, you can run the calculator with the following command:
```bash
./calculator
```
Just input your mathematical expression and hit enter to compute the result.
